# own-repos-curator

リポジトリ一覧を軸に、リポジトリごとの紹介文を編集・管理するTUI。Rustで書かれています。

# 状況
- 開発中

# 用途
- リポジトリ紹介文を効率よく作成したいとき用
  - ※対象は自分の公開リポジトリ（forkとarchiveは除外）

# インストール

Rustが必要です。

```
cargo install --force --git https://github.com/cat2151/own-repos-curator
```

# 実行

```
repocurator
```

# update

```
repocurator update
```

# 破壊的変更
- 頻繁に行います
